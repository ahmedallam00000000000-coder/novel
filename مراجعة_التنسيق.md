# مُراجَعَةُ التَّنسيقِ والإخراجِ لِلنَّشر

*تَقريرٌ شامِلٌ يَفحَصُ المَخطوطَ من حَيثُ البِنيَة، التَّرتيب، الاتِّساق
البَصَريّ، والجاهزيَّة لِلطِّباعَة (نَشر ذاتيّ، بِلا دارِ نَشر).*

---

## ١. الإحصاءاتُ العامَّة

| البَند | القيمَة |
|---|---|
| إجماليُّ الكَلِمات | ~٦٨٬٥٦٦ |
| إجماليُّ الأَحرُف | ~٤١٢٬٢٣٥ |
| إجماليُّ الأَسطُر | ~١٠٬٤٤٠ |
| عَدَدُ الفُصول | ٢٦ (٠ إلى ٢٥) |
| عَدَدُ الأَقسام | ٥ |
| عَدَدُ المُلحَقات | ٧ |
| تَقديرُ الصَّفَحات (A٥) | ~٢٧٠–٣٤٠ صفحة |

> **الحُكم:** الحَجمُ مِثاليٌّ لِرِوايَةٍ تاريخيَّة (٦٠٬٠٠٠ كَلِمَة هي
> الحَدُّ الأَدنى لِلرِّواية، و١٢٠٬٠٠٠ كَلِمَة هي الحَدُّ الأَقصى المُريح).

## ٢. بِنيَةُ الكِتاب

### Front Matter (المُقَدِّمات)

| العُنصُر | الحالة | مُلاحَظات |
|---|---|---|
| صَفحَةُ العُنوان | ✓ | اسمُ الكِتاب + »رِوايَة« + اسمُ المُؤَلِّف |
| الإهداء | ✓ | جَديد، يَتَوافَقُ مع روحِ الرِّوايَة |
| الاقتِباسُ الافتِتاحيّ (epigraph) | ✓ | من كِتابِ المَوتى — التَّعويذة ١٧ |
| الفِهرِس | ✓ | مُحَدَّث، يَشمَلُ كُلَّ المَلاحِق |
| دَليلُ الشَّخصيَّات | ✓ | مُفَصَّل بِالقاهرة وطيبَة |

> **مُلاحَظَة:** صَفحَةُ حُقوقِ النَّشر (ISBN، رَقم الإيداع، اسم النَّاشِر)
> **غَير مَوجودَة بِناءً على رَغبَة المُؤَلِّف** الَّذي يَنشُرُ بِنَفسِه دونَ
> دارِ نَشر. لا حاجَة لَها.

### Body (المَتن)

| البَند | الحالة |
|---|---|
| ٥ أَقسام بِعَناوينَ ثَيمِيَّة | ✓ |
| ٢٦ فَصلًا (صِفر — الخامِس والعِشرون) | ✓ |
| تَرقيمٌ مُتَّسِقٌ بِالفُروقات اللَّفظيَّة | ✓ |
| اقتِباساتُ افتِتاحٍ لِكُلِّ قِسم | ✓ |
| فَواصِلُ مَشاهِد (`---`) داخِلَ الفُصول | ✓ |

### Back Matter (المُلحَقات)

التَّرتيبُ الحاليّ:
١. كَلِمَةٌ من المُؤَلِّف
٢. شُكرٌ وتَقدير
٣. المَصادِرُ التَّاريخيَّة
٤. من بَردِيَّةِ أَحمد الطِّبِّيَّة *(جَديد)*
٥. مَسرَدُ المُصطَلَحات
٦. عن المُؤَلِّف
٧. نَبذَةُ الغِلافِ الخَلفيّ

**اقتِراحٌ اختِياريّ:** نَقلُ المَسرَدِ قَبلَ المَصادِر يُساعِدُ القارِئَ
المُهتَمَّ بِفَهمِ المُصطَلَحاتِ قَبلَ التَّعَمُّقِ في المَراجِع. لكِنَّ
التَّرتيبَ الحاليَّ مَقبولٌ ومُتَّبَعٌ في كَثيرٍ من الرِّواياتِ التَّاريخيَّة.

## ٣. الاتِّساقُ الطِّباعيّ

### علاماتُ الاقتِباس
- مُستَخدَمَة: »الفَرَنسيَّة العَرَبيَّة المَقلوبَة« (»...«)
- ١٬١١٩ مَرَّة `»` و١٬١٣٤ مَرَّة `«` — تَوازُنٌ جَيِّد.
- صِفر استِخدام لِلعَلاماتِ الإنجليزيَّة `"..."` ✓

### الشَّرطَة المُعتَرِضَة (Em-dash)
- ٥١١ استِخدام لِـ `—` (em-dash)
- جَميعُ الـ`-` (hyphen) المُستَخدَمَة كَشَرطَةٍ مُعتَرِضَة استُبدِلَت
  بـ`—` (١١١ تَعديلًا) ✓
- بَقيَت الشُّرَطُ في القَوائِمِ النُّقطيَّة كَما هي ✓

### عَلامَةُ الحَذف
- صِفر استِخدام لِـ `...` (ASCII)
- ٣٢٤ استِخدام لِـ `…` (Unicode) ✓

### العَناوين
- لا يوجَد أَيُّ عُنوانٍ بِدون فَراغ بَعدَه ✓
- لا أَسطُرٍ فارِغَة مُتَتاليَة (٣ أَو أَكثَر) ✓
- لا عَناوينَ بِمُستَوى أَكبَر من `###` ✓

### التَّشكيل
- تَشكيلٌ مُختار يَتَزايَدُ في المَلاحِقِ والمُقَدِّمات
- المَتنُ بِشَكلٍ عام بِأَدنى تَشكيلٍ لِراحَةِ القِراءَة
- هذا اتِّجاهٌ مَقصودٌ ومَقبول

## ٤. ملفُّ Word الجاهزُ لِلطِّباعَة

```
السكريبت: scripts/make_rtl_docx.py
المَخرَج:  غريب_في_طيبة.docx
```

| الإعداد | القيمَة |
|---|---|
| اتِّجاهُ الصَّفحَة | RTL (يَمين-يَسار) |
| الخَطّ | Amiri (احتِرافيّ، مُعَدٌّ لِلطِّباعَة) |
| حَجمُ الصَّفحَة | A٥ (١٤٫٨ × ٢١ سم) |
| الهَوامِشُ المِرآويَّة | داخِل ٢٫٥سم / خارِج ٢٫٠سم |
| تَباعُدُ السُّطور | ١٫٥ |
| المَسافَةُ البادِئَة | ١سم لِأَوَّلِ سَطرٍ في الفِقرَة |
| حَجمُ المَتن | ١٣pt |
| لَونُ العَناوين | أَزرَقٌ لَيليّ (`#1F4E79`) |
| كُلُّ فَصلٍ في صَفحَةٍ جَديدَة | ✓ |
| اقتِباساتٌ مُنتَصَفَة بِمائِل | ✓ |

> **الحُكم:** ملفُّ Word جاهِزٌ لِلتَّحويلِ المُباشِرِ إلى PDF لِلطِّباعَة
> الذَّاتيَّة، أَو لِلرَّفعِ على مَنَصَّاتِ النَّشرِ الذَّاتيّ
> (Amazon KDP، Lulu، Blurb).

## ٥. تَوصياتٌ نِهائيَّة (اختياريَّة)

١. **النُّسخَةُ الرَّقميَّة (EPUB)** — يُفَضَّل تَوليدُها بِشَكلٍ مُنفَصِل
   لِأَنَّ ضَبطَ EPUB يَختَلِف عن ضَبطِ الطِّباعَة. يُمكِنُ استِخدامُ
   pandoc:
   ```
   pandoc غريب_في_طيبة.md -o غريب_في_طيبة.epub --metadata title="غريب في طيبة"
   ```

٢. **التَّقسيمُ إلى مُجَلَّدَين** — إذا تَجاوَزَ الكِتابُ ٣٠٠ صَفحَة في
   الطِّباعَة، فالاحتِفاظُ بِه في مُجَلَّدٍ واحِدٍ هو الأَفضَل (الرِّواياتُ
   التَّاريخيَّةُ تَتَحَمَّلُ هذا الحَجم).

٣. **»نَبذَةُ الغِلافِ الخَلفيّ«** — يُمكِنُ استِخدامُها كَنَصٍّ لِلغِلافِ
   الخَلفيّ حِينَ تُصَمِّمُ الغِلاف، أَو إبقاؤُها داخِلَ الكِتابِ كَمَرجِع.

٤. **الفَحصُ النِّهائيّ بَعدَ التَّحويلِ إلى PDF** — حِينَ تَحَوِّلُ ملفَّ
   Word إلى PDF لِلمَطبَعَة، تَفَحَّص يَدَويًّا:
   - عَدَمِ وُجودِ »يَتيمٍ« أَو »أَرمَلٍ« (orphans/widows): سَطرٌ واحِدٌ
     فَوقَ صَفحَةٍ جَديدَة
   - عَدَمِ انكِسارِ كَلِمَةٍ بِشَكلٍ خاطِئ
   - تَناسُقِ هَوامِشِ الصَّفَحات
   - تَرقيمِ الصَّفَحات

٥. **مُدَقِّقٌ لُغَويٌّ بَشَريّ** — أَنصَحُ بِإجراءِ مُراجَعَةٍ بَشَريَّةٍ
   نِهائيَّة قَبلَ الطِّباعَة. لا لِأَنَّ النَّصَّ ضَعيف، بَل لِأَنَّ كلَّ
   كِتابٍ مَطبوعٍ يَستَحِقُّ هذه الحَفلَةَ الأَخيرَةَ من العِنايَة.

## ٦. الخُلاصَة

> **المَخطوطُ — من حَيثُ البِنيَةُ والاتِّساقُ الطِّباعيُّ — جاهِزٌ
> لِلنَّشر.**
>
> المُؤَلِّفُ يَنشُرُ بِنَفسِه دونَ دارِ نَشر، فَلا حاجَةَ لِبَيانات ISBN
> أَو رَقمِ الإيداع. تَصميمُ الغِلافِ واختيارُ صورَتِه يَعودانِ
> لِلمُؤَلِّفِ حالَ الرَّغبَةِ في إصدارِ نُسخَةٍ وَرَقيَّة.

*تَمَّ إعدادُ هذا التَّقريرِ بَعدَ فَحصٍ تِقنيٍّ شامِلٍ لِبِنيَةِ الملفِّ
ولِكُلِّ تَفصيلٍ شَكليٍّ يُمكِنُ قِياسُه آليًّا.*
